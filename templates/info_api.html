API de balldontlie

La API de balldontlie permite obtener datos relativos a la NBA. Proporciona información detallada sobre jugadores, equipos, estadísticas de partidos, clasificaciones y mucho más. Con una documentación bien estructurada y un pool de datos perfectamente validados y actualizandos, se convierte en una buena alternativa para aplicaciones de este tipo.

Ir a Balldontlie

API gratuita

A diferencia de otras APIs existentes sobre la NBA, la API no requiere suscripción a un plan de pago.

Proyecto de código abierto

Desarrollado por la comunidad, que implementa las nuevas funcionalidades.

No requiere API key

No requiere clave API para su uso. Enorme ventaja.

Estadísticas actualizadas

Contiene datos desde la temporada 1946-actual, actualizados cada ~ 10 minutos.

Limite de solicitudes

Limitado a 60 peticiones por minuto.

Endpoints

Cuenta con multitud de endpoints a los que consultar, con una gran variedad de información.
